# рҹҡҖ How It Works

1. Enter a poem topic.
2. Select the desired mood.
3. Choose a poetic style.
4. Specify the number of lines (except Haiku).
5. The application builds a prompt for the Llama 3.2 model.
6. Ollama generates a unique poem.
7. View, regenerate, or download the generated poem.
